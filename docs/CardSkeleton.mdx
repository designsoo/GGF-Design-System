## Title

CardSkeleton

## Description

The CardSkeleton component represents a skeleton layout for a card, typically used to indicate loading or placeholder content before the actual content is loaded.

## Usage

```tsx
import { CardSkeleton } from 'ggf-ui';
```

```tsx
<CardSkeleton />
```
