## Title

- StarRating

## Description

- The StarRating component represents a rating control allowing users to select a rating value by clicking on stars.

## Anatomy

- **`size`** : Optional string prop specifying the size of the star rating. It can be one of the following values: 'small', 'medium', or 'large'. If not provided, the default value is 'medium'.
  ```tsx
  size: 'small' | 'medium' | 'large';
  ```
- **`rating`** : Optional number prop representing the initial rating value. If not provided, the default value is 0.
- **`readonly`** : Optional boolean prop indicating whether the star rating is read-only. If true, users cannot change the rating value by clicking on stars.
- **`onChange`** : Optional function prop to handle changes in the rating value. It receives the new rating value as a parameter.

## Usage

```tsx
import StarRating from 'ggf-ui';
```

```tsx
<StarRating size="small" rating={4} readonly />
<StarRating size="large" onChange={(rating) => console.log(`Current Rating: ${rating}`)} />
```
