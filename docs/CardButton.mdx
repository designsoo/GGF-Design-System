## Title

CardButton

## Description

The CardButton component represents a button typically used within a card, allowing users to perform specific actions associated with the card content.

## Anatomy

- **`children`** : Content to be displayed within the button.
- **`onClick`** : Optional function prop to handle click events on the button.
- **`color`** : Optional string prop specifying the color variant of the button. It can be one of the following values: `'yellow' | 'red'.

## Usage

```tsx
import { CardButton } from 'ggf-ui';
```

```tsx
<CardButton color='gray' onClick={() => console.log('Button clicked')}>
  Button
</CardButton>
```
