## Title

- Tag

## Description

- The Tag component represents a visual tag used for labeling or categorizing content with different styles.

## Anatomy

- **`children`** : Content to be displayed within the tag.
- **`variant`** : Optional string prop specifying the style variant of the tag. It can be one of the following values: 'default', 'primary', 'secondary', or 'tertiary'. If not provided, the default value is 'default'.
  ```tsx
  variant: 'default' | 'primary' | 'secondary' | 'tertiary';
  ```

## Usage

```tsx
import Tag from 'ggf-ui';
```

```tsx
<Tag variant='primary'>Tag</Tag>
```
