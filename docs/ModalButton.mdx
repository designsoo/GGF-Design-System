## Title

ModalButton

## Description

The ModalButton component represents a button typically used within a modal dialog, allowing users to perform specific actions associated with the modal content.

## Anatomy

- **`children`** : Content to be displayed within the button.
- **`variant`** : Optional string prop specifying the style variant of the button. It can be one of the following values: 'success' | 'warning'.

- **`onClick`** : Function prop to handle click events on the button.

## Usage

```tsx
import { ModalButton } from 'ggf-ui';
```

```tsx
<ModalButton variant='warning' onClick={() => console.log('Modal Closed')}>
  Button
</ModalButton>
```
