## Title

- Badge

## Description

- Displays a Badge with various visual styles to indicate different types of information or statuses.

## Anatomy

- **`children`** : Content to be displayed within the badge.
- **`variant`**: Optional string prop specifying the style variant of the badge. It can be one of the following values. If not provided, the default value is 'default'.
  ```tsx
  variant: 'default' | 'secondary' | 'tertiary' | 'destructive' | 'outline';
  ```

## Usage

```tsx
import Badge from 'ggf-ui';
```

```tsx
<Badge variant='primary'>Badge</Badge>
```
